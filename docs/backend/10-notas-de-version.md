# Notas de Versión Backend

## Versión 0.0.1
- Implementación inicial de API REST con Spring Boot.
- Autenticación JWT básica.
- CRUD para clientes, citas, servicios, productos, categorías, movimientos, pagos, personal.
- Base de datos PostgreSQL con migraciones Flyway (V1-V11).
- Configuración básica de seguridad y CORS.

## Próximas Versiones
- Mejorar seguridad: hashear contraseñas, expiración JWT.
- Agregar validaciones y manejo de errores global.
- Implementar reportes avanzados.
- Tests unitarios e integración.
- Documentación API con Swagger.
- Optimizaciones de rendimiento.
